# Shell Scripts

```bash
git clone https://github.com/ShellTux/shell-scripts.git ~/.local/bin
```

## Pre-requisites

To install dependencies, simply run:

```sh
make install
```

Note: Only works on `Arch Linux`

## List

- arch-wiki - script to launch offline arch wiki pages
- change-volume - API script to change volume that work with different audio servers
- counter - Countdown script with rainbow numbers
- dunst-espeak-critical, dunst notification sound script
- dunst-espeak-low, dunst notification sound script
- dunst-espeak-normal, dunst notification sound script
- grub-playtune, play tune for grub
- layoutmenu.sh
- mpdnotify, watch for mpd track change and notify the current track to the notification server
- notify-music, notify current track to the notification server
- onefetch-languages, print every language for onefetch
- preview, preview script for lf and fzf
- sb-battery
- sb-brightness
- sb-clock
- sb-cpu
- sb-date
- sb-disk
- sb-disk-activity
- sb-internet
- sb-kernel
- sb-mem
- sb-music
- sb-network-traffic
- sb-news
- sb-packages
- sb-popupgrade
- sb-volume
- sb-weather
- scratchpad, launch scratchpad for X11 (dwm)
- screenshot, take a screenshot (x11 only)
- select-emoji, select an emoji and copy to the clipboard
- select-music, select a track on the mpd playlist
- startWayland, launch Window Manager/Desktop Environment Wayland compositor
- startX, launch Window Manager/Desktop Environment X11
- texclear, clear the build files generated by latex compiler
- toggle-mute, API script to toggle mute that work with different audio servers
- wacom, configuration script to set wacom tablet (x11 only)
- wallpaperLoop, just a script to loop over multiple wallpapers
- wallpaper, set wallpaper
- waybar-autoreload, auto-reload waybar watch for config file change
- wl-clipboard-history, wayland clipboard history tracker
